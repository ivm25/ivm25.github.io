Stomach Cancer Awareness in India

Published: Sun 06 January 2019
By me

In posts.

tags: Data Visualisation Data Storytelling

A short Data Science project that has two key purposes:

  1. Improving my data science skills. The best way is to practice and as I am transitioning into data science from academia, I have lots to learn on a daily basis.
  2. My Uncle was detected with a stomach tumour recently, which was growing faster than we could have imagined. It had to be removed surgically before it could morph into a stage 2 stomch cancer. I was a bit shaken honestly and decided to look up for some data on stomach cancer awareness in India. I didnt find one, hence decided to go to google trends and scrape the data Using a package in R.

Step 1: STATING THE ASSUMPTIONS

  1. The Data has been obtained by Web scraping from Google Trends, Which has a limit to the number of data points. Therefroe, the dataset in itself is not large and it might not be representative of the awareness across India.
  2. The data only consists of hits for the Keyword, which can only be an approximate way to measure the awareness of the Keyword.
  3. We will assume here, that Google trends is an reliable way to measure the hits for a Keyword.

Step 1B: STATING THE HYPOTHESIS

"We expect that the awarenes of Stomach cancer will vary according the media as well as a tendency for increased awareness on youtube after 2006, when the digital platforms really took off in India."

Step 2: LAYING DOWN THE GROUNDWORK

It is very important that we lay the groundwork by telling everyone

a) What are we analyzing:

  • We are analzing the stomach cancer awareness in India by means of hits on various medaia. The data has been obtained by web-scraping Google Trends.

b) What our variables mean:

  • There are four catgorical Variables, each of which represent the different media. Geo represents the location (India in this case).

c) Why are we analyzing this data set:

  • The data provides a good way to practice, "Data Extraction", "Data Preparation", "Data Visualisation" and "Data Modelling(Time Series)". Secondly, the data helps addressing a key issue and makes the data analysis process purposeful.

a) What analyses we’ve done on the data:

  • We have extracted basic statistics out of the dataset and found that there no nulls or missing values. We have used datetime method of pandas to convert the dates into datetime format for time series analysis further down the line. The merge fucntion has been used to combine all the different tables into one for a quicker, combined analysis.

    b) Why we did these analysis:

  • To enable better Data visualisations and also extract meaningful insights like changes in awareness with time.

    c) What are the preliminary findings:

  • The highest number of hits (web) are observed for Web and the least for news. It would be interesting to observe when did the number of hits start to decrease for news, as it is a primary source of information for a lot people in India.

Summarizing the results

  • a) In general the stomach Cancer awareness has been increasing in India since the early 2000s.

  • b) World wide web in genral has been the most informative source and has shown a steady number of hits.

  • c) Other channels like Youtube have been instrumental since 2009, in increasing awareness on stomach cancer. Although, the trend has not been consistent and has shown a lot of irregular pattern.

  • d) It would be very useful to predict which media source is most likely to generate more awareness in the near future. Next part of the project would involve some time series modelling.
